# **App Name**: StatTrak

## Core Features:

- Data Input: Allow admin to input student names and grades into the system via a UI.
- Data Storage: Store student records with dummy data in local JSON files to serve as a pseudo-database for demonstration.
- Descriptive Stats: Automatically calculates statistics (average, median, max, min, standard deviation).
- Graphical Representation: Show student grade data and derived statistics using charts and graphs.
- Sortable tables: Display all student and grades data in sortable tables.
- Insights Generation: Leverage AI to analyze the statistical distribution of the students' scores and output reasoning to present meaningful conclusions about overall performance and challenging subjects by employing a tool to decide whether or not to incorporate specific points based on relevance and significance.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to represent stability and intellect.
- Background color: Very light blue (#E8EAF6). It maintains a professional atmosphere while also allowing information to stand out.
- Accent color: Teal (#009688). The addition of teal gives the UI a feeling of vitality and is useful for interactive components.
- Font: 'Inter' for both headlines and body text to maintain a modern, neutral look that suits the analytical content of the website; sans-serif.
- Use clean, minimalist icons from a library like FontAwesome to represent different data types and actions.
- Employ a grid-based layout with clearly defined sections for data input, statistical summaries, and graphical representations. Ensure responsiveness for different screen sizes.
- Subtle transitions and animations when loading data or updating charts to provide a smooth user experience.
